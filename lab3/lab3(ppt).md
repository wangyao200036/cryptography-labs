### 1. Введение в гаммирование

• **Определение**: Наложение гаммы (псевдослучайной последовательности) на открытый текст

• **Историческое развитие**:
  - Схема однократного использования (one-time pad)
  - Гаммирование с конечной гаммой
  - Гаммирование с генератором ПСП

• **Основное преимущество**: Высокая скорость и простота реализации

• **Ключевой принцип**: 

  $$c_i = p_i \oplus \gamma_i$$

  
---

### 2. Математические основы

**Основные операции**:
- Шифрование: $c_i$ = ($p_i$ + $\gamma_i$) mod N 
- Дешифрование:  $p_i$ = ($c_i$ - $\gamma_i$) mod N

**Алфавитное кодирование**:
а = 1, б = 2, в = 3, ..., я = 32, пробел = 0
Модуль N = 33
