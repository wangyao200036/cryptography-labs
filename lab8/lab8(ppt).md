## Основные понятия

- **Большие целые числа**: числа, превышающие стандартные типы данных компьютера
- **b-ичная система счисления**: представление числа в виде $u = u_1u_2...u_n$ по основанию $b \ge 2$
- **Многократная точность**: арифметические операции с числами произвольной длины
- **Алгоритмы**:детерминированные процедуры для выполнения арифметических операций



---

## Алгоритм 1: Сложение

**Вход**: $u = u_1u_2...u_n$, $v = v_1v_2...v_n$, основание $b$
**Выход**: $w = w_0w_1...w_n$ (с переносом)

**Псевдокод**:
1. $j := n$, $k := 0$
2. $w_j := (u_j + v_j + k)$ mod $b$
3. $k := \lfloor (u_j + v_j + k) / b \rfloor$
4. $j := j - 1$, если $j > 0$ → шаг 2
5. $w_0 := k$

**Пример**:$999 + 1 = 1000$ (десятичная система)

---

## Алгоритм 2: Вычитание

**Вход**: $u = u_1u_2...u_n$, $v = v_1v_2...v_n$, основание $b$
**Выход**: $w = w_1w_2...w_n = u - v$

**Псевдокод**:
1. $j := n$, $k := 0$ (k — заём)
2. $w_j := (u_j - v_j + k)$ mod $b$
3. $k := \lfloor (u_j - v_j + k) / b \rfloor$
4. $j := j - 1$, если $j > 0$ → шаг 2
5. $w_0 := k$

**Пример**:$1000 - 1 = 999$ (десятичная система)


---


## Алгоритм 3: Умножение столбиком

**Вход**: $u = u_1u_2...u_n$, $v = v_1v_2...v_n$, основание $b$
**Выход**: $w = w_1w_2...w_{m+n}$

**Особенности**:
- Классический метод "школьного" умножения
- Сложность: $O(n \cdot m)$
- Использует вложенные циклы для обработки разрядов

**Пример**:$123 \times 456 = 56088$

## Алгоритм 4 Умножение столбиком

**Вход**: $u = u_1u_2...u_n$, $v = v_1v_2...v_n$, основание $b$
**Выход**: $w = w_1w_2...w_{m+n}$

**Особенности**:
- Суммирование произведений для каждого $s = i + j$
- Более эффективная организация вычислений
- Снижение количества промежуточных операций

**Формула**:$w_{m+n-s} = \sum_{i+j=s} u_{n-i} \cdot v_{m-s+i}$mod $b$


---


## Алгоритм 5: Деление

**Вход**: $u = u_n...u_1u_0$, $v = v_t...v_1v_0$, $n \ge t \ge 1$, $v_t \neq 0$
**Выход**: Частное $q = q_{n-t}...q_0$, остаток $r = r_t...r_0$

**Ключевые шаги**:
- Нормализация чисел
- Поразрядное деление с коррекцией частного
- Проверка и исправление отрицательных остатков
- Формирование окончательного результата
